### How to build
***only on windows***
* git clone
* mkdir build && cd build
* cmake ..
* open sln in VS
* build RELEASE win32

### How to use
* Copy armadb.dll to Arma 3 root **OR** an addon folder


#### Notes
* SQLite sources from *sqlite-amalgamation-3090200.zip* are included in the *SQLiteDLL* folder
